Resumen: Fedora Linux

Introducción

Fedora Linux, un sistema operativo libre, moderno y seguro. Fedora es como una gran computadora mágica que todos pueden usar, mejorar y compartir. Nació en 2003 y la apoya Red Hat.

Enfoque

  • Innovación: trae lo más nuevo del mundo Linux.
  • Seguridad: protege la compu con sistemas como SELinux.
  • Libertad: promueve software libre y abierto.

Objetivo: ofrecer un sistema moderno y confiable para desarrolladores, estudiantes.

Características principales

  1. Libre y gratuito
  2. Código abierto
  3. Alta seguridad (SELinux)
  4. Actualizaciones frecuentes (cada 6 meses)
  5. Paquetes .rpm y gestor DNF
  6. Soporte Flatpak
  7. Estabilidad a pesar de usar software reciente
  8. GNOME por defecto (se puede cambiar)
  9. Filosofía libre
  10. Base para RHEL

Ediciones (de más a menos importante)

  1. Workstation — para computadoras de escritorio y notebooks (la más usada).
  2. Server — para servidores y redes.
  3. Cloud / CoreOS — para contenedores y la nube.
  4. IoT — para dispositivos conectados (sensores, robots).

Versiones (vieja → nueva)

Primera: Fedora Core 1 (noviembre 2003). Última: Fedora 41 (2025). Total: 41 versiones.

Fedora publica versiones cada ~6 meses; cada versión trae mejoras de rendimiento y compatibilidad.

Lo que hace especial a Fedora (resumen)

  • Colaboración entre comunidad y Red Hat: libertad + calidad.
  • Siempre a la vanguardia con kernels y software nuevos.
  • Buena estabilidad para trabajo y estudio.
  • Defiende el software libre y abierto.

Comparación con otras distribuciones

DistribuciónSimilaridadDiferencia
UbuntuModerna y fácilFedora usa .rpm; Ubuntu usa .deb. Fedora es más "pura" libre.
DebianEstable y seguraDebian actualiza más lento; Fedora es más reciente.
Arch LinuxTiene software recienteArch es Rolling y más técnico; Fedora es más accesible.
RHELRHEL proviene de FedoraFedora es experimental y gratis; RHEL es pago y empresarial.

Lo malo de Fedora

  1. Actualizaciones frecuentes → puede pedir más mantenimiento.
  2. Algunos controladores o códecs no están por defecto (drivers NVIDIA u códecs multimedia requieren repos extras).
  3. No es la más amigable para principiantes absolutos.
  4. Al usar software nuevo, a veces aparecen pequeños fallos.

Resumen final (para el oral)

Fedora es una distribución moderna, libre y segura creada en 2003 con apoyo de Red Hat. Se enfoca en innovación, seguridad y libertad. Ideal para aprender, programar y experimentar. Tiene varias ediciones según el uso y publica versiones cada 6 meses.
